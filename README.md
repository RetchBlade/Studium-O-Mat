# studie-o-mat-vue

## Project setup
```
npm install
```

### Frontend starten
```
npm run serve
```

### Backend starten
```
zuerst module installieren
    npm install jsonwebtoken bcrypt cors mongoose dotenv jwt-decode
starten des backend mit:
    node server.js
```
